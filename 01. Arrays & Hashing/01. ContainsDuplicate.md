# 217. Contains Duplicate
Easy

Given an integer array nums, return true if any value appears at least twice in the array, and return false if every element is distinct.

 

Example 1:
```
Input: nums = [1,2,3,1]
Output: true
```
Example 2:
```
Input: nums = [1,2,3,4]
Output: false
```
Example 3:
```
Input: nums = [1,1,1,3,3,4,3,2,4,2]
Output: true
 ```

Constraints:

- 1 <= nums.length <= 105
- -109 <= nums[i] <= 109

```java
class Solution {
    public boolean hasDuplicate(int[] nums) {
        HashSet<Integer> seenNumber = new HashSet<>(nums.length);
        for (int i = 0; i < nums.length; i++) {
            if (seenNumber.contains(nums[i])){
                return true;
            }
            seenNumber.add(nums[i]);
        }
        return false;
    }
}
```

HashSet is a great usecase here because it has a O(1) for contains and add operations.
Therefore, we can quickly check if the number has been seen before, and add it to the HashSet if it hasn't.
